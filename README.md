# Node.js TypeScript Template

A modern, robust, and minimalist boilerplate for building Node.js applications with TypeScript.

![Node.js](https://img.shields.io/badge/Node.js-v20+-green.svg)
![TypeScript](https://img.shields.io/badge/TypeScript-v5+-blue.svg)
![License](https://img.shields.io/badge/license-ISC-grey.svg)

## 🚀 Features

- **TypeScript 5+**: Configured with `strict` mode for type safety.
- **Modern Syntax**: Write using ES Modules (`import`/`export`), compiles to CommonJS (`require`) for maximum Node.js compatibility.
- **Fast Development**: Uses `tsx` for lightning-fast watches and restarts.
- **Linter & Formatter**: 
  - **ESLint** (Flat Config) for code quality.
  - **Prettier** for consistent code style.
- **Production Ready**: Optimized build pipeline using `tsc`.
